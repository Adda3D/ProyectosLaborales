# BienMio Front
Carpeta del frontend de la app BienMio
## Instalacion
Requisitos para la instalacion:
* NodeJs

Para instalar ejecutar:
``` yarn install```

## Ejecucion
Para la puesta en marcha ejecutar:
``` yarn start``` 

## Para produccion 
* ejecutar ```yarn build```
* comprimir el contenido de la carpeta ````./build```
* Pasarselo al encargado del area de sistemas, en este momento(30/06/2022) Juan Lopez
