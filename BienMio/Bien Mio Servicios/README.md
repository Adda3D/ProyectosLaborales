# ipcc-back-end
Bien mio app backend on node.js 

## Instalacion
Requisitos para la instalacion:
* MongoDB

Para instalar ejecutar:
``` yarn install```

## Ejecucion
Para la puesta en marcha ejecutar:
``` yarn start``` 
deberia mostrar un codigo parecido al siguiente
```
[nodemon] 2.0.15
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,json
[nodemon] starting `node ./bin/www`
App is ready on: Http://localhost:4200
```
## En el servidor
* Actualizar la carpeta de la ruta ```./ipcc-back-end```
